# Responsive Landing Page 

## Project Overview

This project is a modern and responsive landing page developed using HTML5 and CSS3. It includes a professional user interface with a navigation bar, hero section, about section, services section, call-to-action section, and footer.

## Features

* Responsive design for Mobile, Tablet, Desktop, and Large Screens
* Modern navigation bar
* Hero section with Call-to-Action button
* About section
* Services section with responsive cards
* Contact section
* Professional footer with social media icons
* CSS Flexbox layout
* Media Queries for responsive design

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## Project Structure

```
Responsive-Landing-Page/
│── index.html
│── style.css
│── README.md
```


## How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Install the Live Server extension.
4. Right-click `index.html` and select **Open with Live Server**.

Successfully created a clean, modern, and responsive landing page that adapts to different screen sizes and devices.
